# 채팅방 Web Toy Project

## 개요
프론트엔드 HTML, JavaScript, CSS에 대해 공부하기 위해 간단한 채팅방을 구현

## 주요기능
- 사용자 입력
- 채팅 전송
- 채팅 수신
- 채팅 삭제
